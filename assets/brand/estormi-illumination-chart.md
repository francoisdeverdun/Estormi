**Name:** Estormi  
**Line:** Ars Memoriae  
**Tagline:** A private place for everything your life remembers.

## Core Idea

> Preserve every memory. Read it back as one page.

Estormi keeps the medieval-manuscript metaphor for a modern local memory
system. Every source is sealed into a private archive as a durable memory
trace; each day the traces are gathered — by time-window retrieval, not a
precomputed graph — into one illuminated editorial briefing. The visual
language follows from that idea: a blocked illuminated initial leads, the name
reads at any size, and gold is held back so the detail stays precise instead of
decorative.
